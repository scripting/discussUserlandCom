Found why you might have had troubles browsing the Proof of Concept. IE5 is mildly paranoid (although quite rightly) about downloading XML files from different servers that you originally requested the file from.

If you want to view the demo, you'll need to change your IE5 Security settings to Medium-Low or below (either as a global default or through the Trusted Sites group).

In short, if the IE5 security settings are set to:

SettingResult
HighFailure
MediumFailure
Medium-LowPrompt whether you would like to download
LowSilently download

In any case, I've added a quick error message which detects for this -- please try again at http://clippings.cameronmartin.com/xml/ and let me know if it's working for you this time.

Now we've just got to figure out some XML-RPC calls that we'd like you to support without getting too fancy...
