Following up on Dave's comments about a web accessable scanner...

All of the new Xerox "Document Center" devices (ie. multi-function Fax, Copier, Scanner, Printer, etc.) are networked and work EXACTLY like this! You scan in document (which ends up on the devices internal hard disk) and you access all the images via your web browser! In fact, ALL of the controls/functions/etc of the device are accesible via it's built-in "HTTP server". It's pretty cool!

You might also want to look at some new software from Adobe (Acrobat Messenger) which turns an NT box into a "kiosk" scanning station. You put the document in the scanner, press a button on the screen, it scans the document (with optional OCR) and then puts the image and/or text into a new PDF document. The PDF is then accessible either via the web or via Email.
