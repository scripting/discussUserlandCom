If you install IE 4 with Active Desktop then you'll get a favorites menu, even on NT 4. IE 5 does not include Active Desktop but it is built-in to Windows 98 and Windows 2000.

Yes, if you want to have this feature and use IE 5 you must first install IE 4 and then install IE 5.
