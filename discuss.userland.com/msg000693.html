I grabbed the first 690 messages out of the discussion board, converted them to mbox format, and got Kendall Clark to convert them into HTML using MHonArc (http://www.oac.uci.edu/indiv/ehood/mhonarc.html). The server seemed to keep up with the requests pretty well, btw.

A few messages had problems, specifically if there was any non-XML content in the subject line or body. This includes undefined entities and HTML tags.

Go check it out at http://ntlug.org/~niel/discuss/threads.html. Java source code is there too, http://ntlug.org/~niel/discuss/source/.
