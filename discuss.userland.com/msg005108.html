mainResponder.discuss.topicsToXml contains these two lines: 

local (f = mainresponder.discuss.openfile ());
local (adrmsgs = @[f].discuss.messages)

This hardwires adrmsgs to the subtable of discuss.root named "discuss."

I think that it would be better to use mainResponder.discuss.openRoot here to set adrmsgs.
