Another data point: My Frontier Scripting site uses JavaScript client-side includes. Which simply means, they use the src attribute of the script tag to include JS code that uses document.write to add content to the page--on the client side.

The jsInclude(text) macro for Frontier makes it simple to "include" any content you want when you render using Frontier.
