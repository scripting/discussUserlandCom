Can I somehow sort page titles list that outlineSite generates some other way than alphanumerically by page title? I wish I could make it use some other field than #title to sort on.

You might want to try my enhanced version of outliteSite: http://www.inligo.com/frontier/macros/autoDirectory.html.

While it doesn't allow you to specify the sorting field, it does allow you to turn off title sorting so the pages will appear in the same order they appear in Frontier.

I'm also working on a successor to autodirectory which, among other things, will allow sorting by any directive. It's not ready for public release yet, but if you want to be a beta tester, please let me know by email: heath@inligo.com.
