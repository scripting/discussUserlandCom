I'm not sure this is such a good idea. If I reply to a message and the original message is changed afterwards, then the context of my reply is changed without my approval.

Imagine this (a rather stupid example): the original message says "I think Frontier is cool" and I answer with the obligatory "me too" message. Now the author changes the original message to "I think Frontier sucks" or other things a lot worse. See what I mean? If you edit an original message, you automatically edit all follow-up messages.
