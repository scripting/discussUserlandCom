What is the best Linux for using on Mac PowerPC hardware?

You should only run MkLinux on older NuBus machines that aren't supported by LinuxPPC. The LinuxPPC folks did most of the work of porting standard Linux to the Macintosh (MkLinux uses a Mach microkernel beneath Linux), but some folks have been saying good things about Yellow Dog lately.
