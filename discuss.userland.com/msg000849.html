When new pages are added to or changed in the Developers website -- http://developers.userland.com/ -- or verbs are added or changed in DocServer -- http://docserver.userland.com/ -- the search engine is now indexing them immediately.

This means that, for instance, you can find the new page on the Developer's site that's a guide to all the website framework directives and attributes.

The page is here:

http://developers.userland.com/stories/directives

Here's an example search for the "ftpSite directive," which turns up this page as the first hit:

http://search.userland.com/default?m=25&s=1&t=and&q=ftpSite%20directive
