Re "forfeiting all of the value of one's work": In the worst case, I suppose, I could write a program that could have sold for a profit, release it under the GPL, and never get any enhancements back from other developers. In such a case, I've lost the time and effort I spent writing the program. But if no other developer cared about the program enough to enhance it, how much was the program worth in the first place?

Re "undermining one's profession": ESR's essay "The Magic Cauldron" presents extensive arguments for how open-source development can be economically self-sustaining -- and therefore, how releasing source now will not lead to economic ruin later. If you have arguments that address ESR's points, I'd be very interested in seeing them.
