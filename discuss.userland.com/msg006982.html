Is there a particular way I need to write out the links?

It's possible that the parser only recognizes links following a certain pattern, e.g. it may only accept tags in lower case. The script that handles the actual parsing of your news page lifes in tools table of the sample news site, it's called getXmlText.

For a working example, you might want to compare the source of the current Scripting News default page with the following pages:

http://news.userland.com/mostRecentScriptingNews.xml

http://news.userland.com/scriptingNewsToRSS.xml
