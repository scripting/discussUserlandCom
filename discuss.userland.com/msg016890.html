I have written some software that pumps docs from a JetSend to a DMS. The Jet send will do both colour and B&W scanning to TIFF or PDF. In the settings for the unit use the manual colour depth selection and force B&W or Colour if you want any performance. If you are using the automatic setting the unit bogs down trying to figure out what it should use.

The system scans the file to a specified directory. If that directory is exposed through a Web server, Great! The magic happens.
