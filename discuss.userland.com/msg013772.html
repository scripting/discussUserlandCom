A question and a concern:

2. I'd prefer if you didn't read the file every minute. That's an unnecessary load on our server. The file will change sometime between seven and eleven minutes after the hour. Esp since you're distributing the code, it would work better all-around if there weren't a lot of unncessary hits on our server.
