remove floppy disks from the drive when you're done using them (so that you don't accidentally boot from one)

A more permanent solution is to visit the bios setup utility (hit del during memory check on most PCs) and set the default boot device to IDE0 (the primary hard drive). During normal use these days, one practically never wants to boot a floppy - when you do, just go back the bios and switch it again. This way floppies left in the drive can do no harm.
