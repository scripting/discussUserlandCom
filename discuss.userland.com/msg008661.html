This context-based search is not new - http://www.scoobs.com has been offering it for several months and I gather from Walter Underwood, Infoseek's technical guru, that their next search product will as well. Actually Scoobs is open source, so anyone can put it on their site or use it.

The other problems are that this search engine requires you to register one page at a time (and only manually - not like the search.userland engine which allows submission through an xml-rpc interface), and currently xml documents are not heavily hyper-linked like html ones are, so growth of the index is going to be slow. Of course when xlink arrives....
