While pondering the issues involved in managing workgroups who are collaborating via both the DG and Web-Edit, it occured to me that there ought to be some link between the People Suite and the membership groups.

Does such a link yet exist?

It would not seem too difficult, if not, to create a link where people.newGroup(), people.newUser(), etc. would also maintain a membership group and set of users. The administrator could then define both DG and WebEdit access for workgroups from one interface.
