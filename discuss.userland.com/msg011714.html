This is a feature for channel developers.

If you make a change to your channel that you want to be immediately reflected on all the affiliate sites, just visit this URL:

http://aggregator.userland.com/reloadChannel$2765

Replace 2765 with the ID of your channel.

This forces your XML file to be read immediately, it's compiled, the html rendering is recalculated and any new stories are sent to the affiliates.
