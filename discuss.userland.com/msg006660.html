I believe Netscape has this feature since they shipped Netscape Gold (now called Netscape Composer). Just browse to the page you wanna edit, and choose "Edit Page" from the File Menu, and you will be able to edit the page (and see all the graphics as well). Then when you choose "Publish" from the File Menu, the page will be sent back to the server. You can also use an External Editor to edit HTML source and the images.

The interface is far superior than what Dave has talked about, but I have never truly used it. There are many times when working locally is a much better choice, like when you need to do a batch search and replace.

BTW, I saw a screen shot of the coming Mac OS (8.7). With the new version of the Navigation Services, an app can access a web server or a FTP server on the system level!! Of course, we probably wouldn't see it appear on Frontier. Frontier doesn't even have contextual menu, neither on Mac or PC :-(
