The Windows Installer, which will (hopefully) be built in to Windows 2000 seems to support just-in-time installation pretty nicely. You have to somehow get a foot in the door, though, to let it know that a particular component is available.

I think it will still matter which components are and are not on the Windows CD-ROM in the future. If I buy a Windows 2000 Workstation upgrade, I had better see every free MS product (that fits) on the CD-ROM so that I don't have to download them all.
