We have a simple spec for the Search Engine interface that we use for our server. The docs are on this page:

http://developers.userland.com/stories/searchEngineInterface

What I'd love to see is a search engine implemented on Linux connected to MySQL, that supported this interface.

We would be very interested in running that software here at UserLand to supplement our own search engine software which is designed for smaller indexing jobs.

This seems like an ideal collaboration between CMS developers such as UserLand who work on Windows/Mac and developers who work on Unices.

Would anyone like to work with us on this at this time?
