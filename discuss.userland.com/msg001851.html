Good evening. I'd like to announce telnetServer 1.0a5, available now at:

http://www.john.pair.com/telnetServer/

This release adds Frontier 5 support, including Frontier 5 for windows. A new user manager command is included, negating the MacBird interface.

What is telnetServer

It's a telnetServer for Frontier. It allows you to telnet into your Macintosh or Windows computer and browse both the file system and frontier.root.

It sports a command line interface, and most typical unix commands are supported. See

http://www.john.pair.com/telnetServer/docs/executingcommands.html

You'll also need to download the regex DLLs from:

http://www.scriptmeridian.org/projects/regex/

I should mention that while it says alpha, this is fairly well tested and burned in.
