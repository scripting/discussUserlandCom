DRRHO-xml-rpc-0.1

handles XML-RPC request for Apache/Mason

http://www.masonhq.com/arch/viewItem.html?author=DRRHO&handle=xml-rpc

From the documentation:

This component handles XML-RPC request against the server. It takes no field parameters, but a single XML stream via POST as described in the XML-RPC documentation. It then calls the RPC dispatcher and returns the result XML to the client.
