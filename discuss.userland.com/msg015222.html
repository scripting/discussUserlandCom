How would it work?

My idea (for what it's worth):

Update Manila.root so that when any Manila website is created, generate a glossary entry with name == the site name (from mySite.["#newsSite"].siteName) and give it the value of the url (stored at mySite.[#ftpSite"].url) wrapped in an anchor tag around the site name. All Manila users would then automatically get a "My Site" shortcut.

If the Manila website is created in isp.root (like all the sites on ETP.com) use the global glossary instead of the site's glossary. ETP.com users would then automatically have access to shortcuts for all other ETP.com sites.
