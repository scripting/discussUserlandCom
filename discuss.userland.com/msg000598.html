How would you like to have a discussion group that didn't only display in HTML, that you could build a custom interface for in any application that had an HTTP client and an XML parser?

Well, we just took the first step in that direction today. You can now read any message on this board in XML form. Here's what a URL looks like:

http://discuss.userland.com/msgReader.xml$598

Just change the number after the $ to get a different message.

Onward!

Dave
