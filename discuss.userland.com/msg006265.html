First, check out the new preferences system, and if you haven't already done so, fill in your name, a link to your personal website, etc.

http://prefs.userland.com/

All this stuff helps make your DG postings more interesting. It's nice to have a name and a bunch of other information to go with each message you post here. (To the newbies, when you click on a person's name in the message header you can learn about them.)

Developers

Now, if you're a Frontier 6 developer, and if you are using membership and are storing preferences for each user, please download prefs.root from the end of the Examples page:

http://frontier.userland.com/titleSearch$examples

It's very easy to configure, all done thru simple XML in the outliner. It also implements JavaScript-based popup windows, the first sample code from UserLand that does this.
