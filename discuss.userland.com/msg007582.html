Do you debug Frontier with Spotlight (on the Mac) or Purify (on Windows NT)?

These tools are absolutely wonderful for detecting memory leaks, file leaks, writing off the ends of arrays, etc. and can do wonders for improving an application or server's stability.

PS - It's really cool to hear that even the latest version of Frontier will run on versions of the OS back to System 7. Old 68K boxes make great servers!
