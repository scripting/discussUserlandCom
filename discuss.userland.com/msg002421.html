You don't have to strip the HTML tags before sending your page -- the search engine will do so before indexing.

But -- you might want to anyway, to make the XML-RPC request a little smaller.

If you want to strip HTML tags, the recommended verb to use is searchEngine.stripMarkup.

I just put up a DocServer page for it:

http://docserver.userland.com/searchEngine/stripMarkup
