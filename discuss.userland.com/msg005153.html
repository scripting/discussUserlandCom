LibGd is a tool for creating Gifs on the fly. It's at http://einstein.stcloudstate.edu/libGd/.

Andre pointed out that Gif strings should be returned as binary of type 'GIFf' instead of as string type. This might be important if you are going to use LibGD with mainResponder or the website framework. Version 1.0a10.1 implements this change.
