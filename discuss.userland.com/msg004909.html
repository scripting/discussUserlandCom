Erik,

Which welcome page are you using? Did you copy it from logon.welcome in htmlInterfaces.root?

If you look in the welcome page in htmlInterfaces.root you'll see the links are set up to go to

responderattributes.urls.discussHomePage responderattributes.urls.prefsHomePage responderattributes.urls.memberLogoff

If your using these default values in your welcome page you'll need have these setup in your #urls table. You can cut-and-paste them from config.mainresponder.urls and change the server/path info.
