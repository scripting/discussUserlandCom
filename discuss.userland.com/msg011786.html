If you get "sendmail" up and running on Linux, then adding XMLRPC interfaces would be pretty easy using the Net::SMTP and Net::POP3 perl modules on an Apache/modperl installation.

For those who aren't familiar with modperl, it's an extension to Apache that includes a Perl interpreter. This gets rid of the overhead of loading PERL and compiling the scripts every time. Very fast!

This would indeed be an interesting project. Are you interested in contracting out any of this work? *grin*
