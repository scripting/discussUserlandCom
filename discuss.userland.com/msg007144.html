A thread is going on perl5-porters about this topic. The basic proposal is to allow (example in Perl): open(IFH, "http://davenet.userland.com"); as well as the conventional: open(IFH, "h:/metapage/userland/davenet/index.html");

This could be extended through the under-used HTTP PUT directive, which I think will become much more useful now that WebDAV is here. Specifically, to write a Web page one would do (in Perl): open(OFH, ">http://davenet.userland.com"); (the Perl syntax for creating/overwriting a file).
