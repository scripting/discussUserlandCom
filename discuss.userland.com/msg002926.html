What a user might want to know about IdleTime:

As a user:

Who is it from ? (The Extensions Manager can show you if you put the answer in a resource.) What stops working if I remove it ?

As a geek:

Idle hooks tend to eat resources on the Mac. Is it PowerPC native ? Does it patch traps ? How much system heap does it consume, and will it not consume memory if nothing currently is using it ?
