I am afraid not. If you had mainResponder installed, I don't think you would still have a user.webserver.responders.default.data.docTree table.

Here's something to check -- user.webserver.responders.default.methods.any should be a script looking as follows:

on any (adrParamTable) {
	return (mainResponder.respond (adrParamTable))}

If it looks different on your machine, you probably haven't run the mainResponder install script. Check the ReadMe.html file in your Frontier 6.0 folder for instructions on installing mainResponder -- look for the Guest Databases Folder heading. Is that it?
