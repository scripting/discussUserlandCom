I put up a Frontier converter for XML-RPC tagged text to WDDX tagged text.

http://209.181.141.42/frontierMirror/workspace.wddx.sit.hqx

http://209.181.141.42/frontierMirror/workspace.wddx.zip

I don't think it has any bugs, but there might be, this is my first go at it.

The script at workspace.wddx.convertToWddx takes one parameter, a string of tagged XML-RPC text minus HTTP headers.

It returns a string, tagged WDDX text.

This script has a test code bundle so you run it right off the bat and get a small demo. It converts workspace.wddx.xmlRpcSample to WDDX text.
