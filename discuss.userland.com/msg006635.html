First -- use the Unlock Semaphores menu item to unlock that semaphore.

Second -- switch from outline or text file logging to gdb logging. In other words:

1) Set user.log.prefs.flLogToOutline to false.

2) Set user.log.prefs.flLogToTextFile to false.

3) Set user.log.prefs.flLogToGuestDatabase to true.
