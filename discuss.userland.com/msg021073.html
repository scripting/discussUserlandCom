Anyone running a Filemaker Pro web server has access to the user passwords. This is not uncommon practice for webservers. There is certainly a difference between the security required for file sharing vs. access to a website.

Any user would be smart to not use the same password for their online banking as they use for other less secure sites such as weblogs.
