A visited link gets its colour, when the page to which it refers, is in the history list of the browser.

Is there a possibility to give a link an aditional colour, when the page to which the link refers, is updated after the last visit?

Shouldn't there be a feature to send information to browsers history lists, when the users have enabled that feature?

I think an additional link definition would be needed as well as a way to submit the information. How could it be handled with static pages?

I have posted this request at http://lists.w3.org/Archives/Public/www-html/2000Jan/0207.html but wasn't contented with the reactions.
