In websites you want to render statically, set #imageExtensions true.

For dynamic sites, set #imageExtensions false.

Remember you don't have to set this in user.html.prefs, you can set it in the #prefs table in your websites, so that different websites have different preferences.

Note that dynamic and static sites have different preferences regarding image extensions.
