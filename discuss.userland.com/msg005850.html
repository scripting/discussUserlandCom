By Apple's own admission (see the MRJ-Dev mailing list), JManager is very difficult to implement properly. They recommend you call Mac code from Java rather than vice versa, which isn't very useful in the case of Frontier. None of the Mac browsers do a perfect job of JManager support; iCab and the Netscape plugin do a much better job than IE.

That said, there's a nice lightweight browser for Java called the ICE browser.

<http://www.ping.be/beta9/more.html>

Ignoring the memory usage of a Java VM for the moment, this could work.
