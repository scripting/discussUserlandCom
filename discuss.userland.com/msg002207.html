What is Tonto?

Tonto is a small helper application for Frontier. It puts up a small floating window, which displays buttons in any application, for activating Frontier scripts. Mac Only

The floating window is available in all applications, and the buttons shown in the window can be global (all apps), context-sensitive (specific to the frontmost app), or a combination of both.

http://www.macrobyteresources.com/scripting/frontier/tonto/

This Release

This version of Tonto has improved support for Frontier 4, fifteen new icons designed by James Spahr (for use in your buttons), and an improvement in the way buttons are drawn (button width varies depending on the width of your captions).
