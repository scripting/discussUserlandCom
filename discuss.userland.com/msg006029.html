Here's an RDF fragment which could be used to describe relocation:

How it Works

This chunk of RDF translates into the following English:

The resource http://www.bar.com/oldpath/news.rss has been RelocatedTo the resource http://www.bar.com/news.rss.

RDF is used to represent statements about resources. You can read more about it at the W3C if you need a good remedy for insomnia.
