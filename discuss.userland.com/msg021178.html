The Magical Code Tree has borne fruit!


We are pleased to announce the release and immediate availability of
BBEdit 6.0.


Major improvements in this version feature:


*   full support for editing multi-byte (for example, Japanese) text,
    as well as Roman text stored in Unicode files;


*   enhancements to the HTML markup tools to support HTML 4.01, XHTML
    1.0, and WML 1.1;


*   enhanced HTML syntax coloring and function navigation, now
    including colors for attribute names, attribute values, processing
    instructions, and embedded PHP3;


*   advanced file filtering for multi-file search (and replace);


*   developer-extensible syntax coloring via a plug-in module
    architecture;


*   significantly enhanced OSA support, with robust Apple Event Object
    Model support, pervasive scriptability and recordability, and menu
    attachability;


*   a beefed-up Glossary, featuring multiple-set capability with
    language-sensitive automatic switching of glossary sets;


*   quoted-text manipulation tools, ideal for preparing email and
    Usenet postings;


*   Appearance Manager awareness;


*   Navigation Services support;


*   ...and bunches of other goodies!


Full product information, including a demo and upgrade information for
current customers, is available on our Web site at
<http://www.barebones.com/products/bbedit.html>.


The full text of the BBEdit 6.0 press release may be found at
<http://www.barebones.com/press/releases/2000/09.07-bbedit_6.html>.
