Announcing Ocha Suite & News Room v1.0b7

Ocha Software is pleased to announce a major revision in the Ocha Suite beta process. This package of tools is designed to simplify and enhance website management in Frontier. The goal was to create a modular system that adapts to the requirements of any particular site. Ocha v1.0b7 is free software (GPL open source) compatible with Frontier 4 and higher for Macintosh and Windows.

http://www.ocha.net/suites/ocha/

Highlights from this release include:

  • Compatibility with Frontier 6 (untested).
  • News Room v1.0b7 including a new browser shared menu feature on the Mac.
  • Includes Enhanced imageRef Macro (EIR) v2.2 with recently updated docs.
  • A new time tracking utility that utilizes menu commands and the Frontier status window to help you keep track of how much time you've been spending on each project/website.

There have also been a large number of miscellaneous changes and enhancements in the past few months since the last release. The documentation has not been updated for this version because we expect some more changes to the menu structure before too long. Your help and participation is appreciated. Thanks!

Jeb Bateman
