In terms of the recent discussions about improving the in-browser editing experience for writers, check out the following site:
http://www.ephox.com/editlive/
The EditLive site has a demo page but you need to install the plug-in first.
http://www.ephox.com/editlive/demos/homepage/default.asp

Ephox EditLive! is an ActiveX control and Netscape plug-in that allows Web authors and application developers to add WYSIWYG HTML editing capabilities to their Web sites and web-based applications. The plug-in uses standard JavaScript & DHTML technology to provide access to editing services such as HTML formatting, tables, images, file handling and more.

Disclaimer: I do not work for this company or know anyone who does.
