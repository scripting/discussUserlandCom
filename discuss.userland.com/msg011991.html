Ability to re-use code is key

Author:Brett Glass
Posted:10/13/1999; 7:11:39 AM
Topic:Today's scriptingNews Outline
Msg #:11991 (In response to 11973)
Prev/Next:11990 / 11992

Open source which programmers can re-use to build something better and sell it prevents the sort of mediocrity you mention from happening. The various BSD UNIX implementations are licensed in this way. Alas, Linux is licensed under the GPL, which attempts to force developers to give their creative work based on the original code away for free. This is a strong disincentive to progress and encourages mediocrity and/or reinventing the wheel needlessly.

--Brett Glass
