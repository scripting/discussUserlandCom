I see it's a W2K server. I experimented with one of the later betas. With just a basic installation the oS used up almost 100MB of RAM and 1GB of hdd. Which means that the least amount of memory in a W2K server should be 256MB, more or less the same as an NT4 box with 128MB of RAM. A heavily used server should therefore have 384 to 512MB of RAM as a start.

And the OS partition ought to be between 1.5 and 2GB.
