this is my personal christmas present to the userland community.

at http://virtuacom.com/wap you can download a fine WAP-emulator for WinOS and PalmOS. Albeit this software is still in beta it is free and works fine. just to illustrate how easy it is to use WAP ( WML) rendered dynamically out of Frontier i implemented the same code (slightly changed) Dan Garner did with ASP in Usertalk. Pretty easy.

So when you've dowloaded the WapMan.exe start it,click the GO-button of the floating browser-window and type in the following address: http://on1.zkm.de/hellowml

WapMan is nice because you need no GSM Phone with WAP-capability.
