At WWDC I learned that CoreFoundation (one of the C APIs for MacOS X) has built-in support for XML, and that OS X's preferences will be stored in XML format in the user's preferences folder.

It appears the CFDictionary type (key/value pairs) can write itself out as XML, and create itself from XML.

Good deal!
