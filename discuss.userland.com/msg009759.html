But mainResponder is adding a closing tag to the end of messages.

1. Make sure your mainResponder.root is up to date.

2. Jump to config.mainResponder.prefs.legalTags.

3. Delete your entry for img.

4. Create a table named img, at config.mainResponder.prefs.legalTags.img.

5. Create an flClose boolean in that table and set it to false.

6. Create an flLegal boolean in that table and set it to true.

7. Save config.root.

Now img tags are legal, but they won't be automatically closed.
