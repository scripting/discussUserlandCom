Here's the code that removes NSI's offensive message:

local (i, ix);
whoResult = string.replaceAll (tcp.examples.whois (domainName), "\n", "\r\n")
for i = 1 to 3 //pop off the first three lines
	ix = string.patternmatch ("\r", whoResult)
	whoResult = string.delete (whoResult, 1, ix + 1)
