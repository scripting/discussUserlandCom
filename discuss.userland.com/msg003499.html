If you make a new string value in a table, putting one space in enters the value 'true'

If you want just one "real space", use option+space

In this case, you don't have a "real space" but an "un-breakable" one (&nbsp;). If you want a real space, type it inside quotes. The quotes will be removed when you validate the cell and the space will stay.
