I think you'll find most people in the LoC world are currently using Z39.50 based technology. There is a move to provide an XML encoding as an alternative to the current BER ASN.1 encoding, but I'm not sure how far along this is.

I'm not aware of any general move towards XML DTDs and away from the current Z39.50 attribute sets such as BIB1, GILS, CIMI, etc.
