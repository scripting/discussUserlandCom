Summary, convert all entries in a table from type of string to type of address.

Use a simple script to walk the table: 

on ChangeStringToAddress( tableAdr )
   local ( idx )
   for idx = 1 to sizeOf( tableAdr^ )
      if ( typeOf( tableAdr^[idx] ) == stringType )
         try
            tableAdr^[idx] = address( tableAdr^[idx] )


ChangeStringToAddress( @myUsers )
