It's at http://einstein.stcloudstate.edu/libGd/.

LibGD is a Frontier graphics extension (DLL) for creating and manipulating images in GIF format. It includes a text engine for rendering text with TrueType outline fonts.

This last version produces somewhat more compact image files than earlier versions. Since I'm planning no further updates - except for bug fixes, I'm calling it a beta.
