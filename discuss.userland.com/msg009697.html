I have a copy of Hooked On Java, done by members of the Java Development team. It lists these people as the development team:

Chuck McManis, Chris Warth, Herb Jillinek, Tim Lindhold, Arthur van Hoff, Michelle Huff, Jonathan Payne, Frank Yellin, Patrick Chan, Erik Gilber, Eugene Kuerner, Mark Scott Johnson, Richard Tuck, Lisa Friendly, Sami Shaio, Bob Weisblat, James Gosling, Kim Polese and Kathy Walrath.

Joy is thanked in the acknowledgements page, but I didn't see any other reference to him.

Of the authors, van Hoff is credited with co-design of the Applet API (with Shaio) and the author of the Java compiler. Shaio is also credited with the user interface tookit and most of the security mechanism. The third author, Orca Starbuck is not shown on the developer list, but credited as a technical writer for the project.
