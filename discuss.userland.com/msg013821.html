if you look closely at the doubleclick/netgravity tag, you can see that they use both IFRAME for IE users and javascript for Netscape users.

One thing that you don't see is that the sure fire way to avoid cacheing a http response is to set a cookie in the response.

so, to try to get your javascript to be reevaluated each time, try setting a cookie in the javascript function.
