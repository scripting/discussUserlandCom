Maybe there are two areas to focus on.

Author:Bill Humphries
Posted:2/12/1999; 6:48:08 PM
Topic:XML for weblogs
Msg #:2867 (In response to 2836)
Prev/Next:2866 / 2868

The first is representing the meta data, such as elements that carry the date and time, the keywords describing the entry, the source, the URLs referenced, etc.

Then there's the descriptive information about the entry. Which, as Jorn points out, is the hard part since some news pages/web logs are highly free form.

Can we get some value right away by agreeing to standards for metadata and wrap free-form HTML description/comment data in an element mapped to the HTML namespace?
