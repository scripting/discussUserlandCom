About relative links and portability -- when you take a Manila site on one server, and move it to another, and use the Install Site... command in the Server menu, your URLs will be updated. This includes absolute URLs in all discussion group messages.

Say I had a site at http://server.com/foo/

If I move it to http://server2.com/bar/ the absolute URLs will be updated.

If I had a link to http://server.com/foo/myStory it would be changed to http://server2.com/bar/myStory

Check out manilaSuite.admin.changeSiteUrl, which does the URL replacing.

The upshot is that absolute URLs are okay for portability.
