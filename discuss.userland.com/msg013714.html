You serve contents with any HTTP server (we are serving with Frontier), in wml format.

Between your server and the client there usually is a WAP Gateway (there are some in development, we are using a beta of Nokia's one) that bridges the gap between http and your (usually wireless) device.

So it's:

HTTP -> Wap Gateway ->[wireless connection]-> Cellphone/PDA

The problem seems to be that not all WAP Gateways work with al WAP devices: the Nokia gateway works well with Nokia cellphones but not with other brands.
