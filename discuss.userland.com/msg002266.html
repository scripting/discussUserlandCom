I have uploaded the new version 0.02 of the Frontier-RPC Perl module to <http://bitsko.slc.ut.us/~ken/xml-rpc/> and it will be finding it's way around CPAN shortly.

Version 0.02 contains the following changes:

- support for, and requires, the later versions of XML::Parser (2.19 is the current version)

- added a simpler client demo, `examples/states-client.pl'

- updated mailing list information and required Perl module in the README

- added a QUICK USAGE GUIDE to the README

- Frontier::Client now takes a `url' argument instead of `host' and `port' arguments

- fixed a missing `Content-Type: text/xml header'
