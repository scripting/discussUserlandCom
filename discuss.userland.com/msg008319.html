Here's something I've been toying with.

I have a dynamic channel description at

http://myhome.mystuff.net/myweblog/search.news.yahoo.com.cgi

this channel takes uses the query string to dynamically tailor the channel to the request.

for example:

http://myhome.mystuff.net/myweblog/search.news.yahoo.com.cgi?netgravity

will return the search results for "netgravity"

the interesting upshot of this is that in conjunction with a "weblog" interface, it becomes a change notification mechanism. any new hits on the search will show up in the log, and old ones will remain in the log history.

i'm very intrigued by this kind of dynamic syndication. any other applications?

COmments?
