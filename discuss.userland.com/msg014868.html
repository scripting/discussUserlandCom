Web Serving for Cheap

http://www.webreview.com/pub/2000/01/28/feature/index2.html

Setting up a MkLinux server is easier than you might think ...

• Dust off an old Mac 6100, 7100, or 7500.

• Download a disc image of MkLinux.

• Partition and format the hard drive.

• Install MkLinux.

• Configure the system and fire up the Apache web server.

• Set up permissions and directories.

• Serve pages.
