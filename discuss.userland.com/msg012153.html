Murray Pearson asks:

I use WebEdit to create lots of sites that I serve statically or semi-statically, rendering them on the server. I do this (rather than using FTP or opening the server volume via a script) because I keep CGIs in the website table and install them as part of the rendering process.

Problem is, every time I render a page or table, ^$^&#$%^$#^ Frontier launches the web browser to show me its handiwork. Running a browser on a web server is a big no-no, I have heard. I have searched high and low for a preference to turn this previewing feature off completely, to no avail. Can anyone help me?

The answer is: set #displayEachRendering to false.
