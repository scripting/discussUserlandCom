In VBScript, if I sink an event, I can't return values in ByRef parameters unless the ByRef parameters are Variants. This limits VBScript's usefulness as a COM client, because lots of ActiveX objects have events with strongly typed parameters (as is recommended by Microsoft in the ActiveX Component technology guides, etc).
