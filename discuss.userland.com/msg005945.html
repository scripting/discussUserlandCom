Actually, this may have been in MSIE4, I'm not sure.

With a single click you can select an entire block-level chunk in the browser window by first holding down the control key and then clicking within the block.

This makes it easy to copy units of text, such as code blocks within PRE tags, HTML lists, etc.

Not an earth shattering feature, but convenient.
