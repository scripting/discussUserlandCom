If you use the server on dolphin.calendar.com, port 8000 and ask it for a latlong it doesn't know, it will now return a fault response. In the Python client lib it correctly raises the xmlrpclib.Fault exception.

Just to summarize the interface, latlong can be called as

    latlong(city, state)	# US or Canada
    latlong(city, "", country)	# other countries
