IE default Fonts: Control Panel, Internet icon, General Tab, Font button.

Showing file extensions: Open up MyComputer, View Menu, Options Menu Item, View Tab, Deselect Hide file extensions

Default Browser: Control Panel, Internet icon, Programs Tab, Specify your default e-amiler etc., Check the box at the bottom of the tab to have IE check to see if it's to be the default browser.

I've got both N4.5 and IE5b2 on my machine, both cooperating.
