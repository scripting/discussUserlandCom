Security Alert: htmlInterfaces.root

Author:André Radke
Posted:4/22/1999; 6:31:36 AM
Topic:Security Alert: htmlInterfaces.root
Msg #:5184
Prev/Next:5183 / 5185

We have identified a security hole in htmlInterfaces.root, a guest database which ships as sample code with Frontier 6.

You can potentially be affected by this security hole if

* You are running Frontier 6 as a webserver or behind another webserver

* mainResponder is installed on your machine

* htmlInterfaces.root or discussion groups based on it are accessible from the web

Information on how to close the security hole is available from the following page:

http://frontier.userland.com/security/htmlInterfacesRoot
