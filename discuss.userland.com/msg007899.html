OK, just keep things moving, how does this strike everybody? Links are inline tags, and there's a bunch of extra metadata to grow into. Like it? Hate it? 


  
    <link url="foo">Here's</link> the title
  
  
    And here's some more text with a link in it.
  
  The Times
  http://www.thetimes.com/<;/sourceurl>
  TestUrls
http://thecategory.com/definition.xml<;/categoryurl>
  Mon, Mar 09, 1999 00:45:29 GMT
  Sun, Mar 15, 1999 00:00:00 GMT

text - mandatory. contains all non-headline text. 1 per item.
title - optional. denotes headline status. 1 per item. Precedes text.

I'd say the following could be attributes of , but there are so many...
source - name of the publishing entity, if not local. Optional.
sourceurl - url of the publishing entity, if not local.
category - predefined category, if available.
categoryurl - where to get a definition of the category
pubdate - date published. optional.
expdate - date the item becomes invalid; optional.
