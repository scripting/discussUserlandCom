I believe that the problem is that you're using the JavaScript event handler onkeyup, which I don't think that IE for Mac supports.

You can test that out at http://www.chalcedony.com/scripts/userlandtest.html, which uses the onkeyup handler without any Java.

In terms of Java talking to JavaScript and vice versa, that's straightforward. Check out either Chapter 9 of "JavaScript for the World Wide Web: Visual QuickStart Guide, 2nd Edition" or Chapter 11 in "Java for the World Wide Web: Visual QuickStart Guide". The former discusses the JavaScript side and the latter the Java side of the equation. I'll admit up front that I'm biased and wrote both, but it's very useful and not difficult.

You can find out more about the Java book at http://www.chalcedony.com/java/ and more about the JavaScript book at http://www.chalcedony.com/javascript/.
