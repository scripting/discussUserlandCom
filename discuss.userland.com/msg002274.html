I have a Perl server running the states demo at `http://ntlug.org:8000/RPC2'.

The methods for the states demo are `examples.getStateName', `examples.getStateList', and `examples.getStateStruct', which take a single value, an array, and a struct, respectively, as parameters.

This server also has an additional method called `echo' which returns an array containing all the parameters you send it.

It also appears that there's a bug in the Perl client that returns errors as a number 3 instead of raising an exception, I'm checking into that.

It also appears that `http://betty.userland.com/RPC2' is no longer available, is there another simple Frontier RPC server available?
