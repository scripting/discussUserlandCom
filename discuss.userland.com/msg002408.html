One thing that I think is important about Frontier is the fact that UserTalk is a sane programming language (as opposed to, say, Visual Basic). Also, it's pretty unlikely that you'll run into something in Frontier's Website framework that isn't scriptable or modifiable, because it's almost all scripts. Thus you won't run into any arbitrary limitations.
