The crucial thing is that "virtual objects" like "/a/b/c.html" need to have everything above in the path (the folders "a" and "b") defined as category. I used to think that I can just map a story to *any* path, no matter how long, but that doesn't seem to work.

That's right. It's very close to working with actual folders and files. If you want a file available at /sub1/sub2/sub3/file, you need to create folders named sub1, sub2, and sub3. Same thing here, you need to create categories.
