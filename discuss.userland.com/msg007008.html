the problem is that now my calendar links lead to the following address: http://news.zkm.net/news.root/1999/06/04 instead of http://news.zkm.net/news.root/sampleNewsSite/1999/06/04

First, you can drop the ".root" part from the URL, like so:

http://news.zkm.net/news/sampleNewsSite/1999/06/04

Second, to fix the calendar links, add an entry named calendarRootURL to the #prefs table. It's value should be the URL of the news site table, e.g.

http://news.zkm.net/news/sampleNewsSite/
