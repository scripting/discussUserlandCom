Slashdot has an article about some guy who's setting up a patent market. The article is available from business2.com.

Visit the site and looking in the lower-right corner of the browser. There's a widget that allows you to evaluate the content on a 1-5 scale! Kind of neat.

I chased the links down, and made my way to the OnlineOpinion FAQ. Apparently you can use this on any site you want to gather data. It's definately targetted as business; the first few reports are free, then you're looking and hundreds/thousands to continue. Detailed pricing information.

Egads! look at this snippet from the installation instructions:

1. Open the html file on your web server

2. Paste the code at the end of your HTML document. NOTE: The code can appear at the very end, even after the tag.

Oi! Not only do they espouse adding the code to every page by hand without even mentioning that you should consult your documentation/support staff to see if there's a better way, they ask you to violate HTML standards too. "qbullet.smiley"

At any rate, it obviously could work on any Manila site with no added Frontier/Manila plug-ins or assistance from the site operator.

This looks like it could be useful to some people.
