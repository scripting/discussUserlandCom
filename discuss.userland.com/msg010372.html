If Apple is not already providing an executive to oversee the Java development in parallel with Sun, it needs to do so, IMHO.

Also, Apple should be working with IBM and in particular with alphaWorks to make sure those companies release products that can be used on Macintosh computers. Usually, those products make the really bad assumption that only non-Mac people will be using their Java software. I recently found the installer for alphaWorks' "Explorer" product failed to install on my Mac.

It is really about time that Java software developers make sure that when they indicate that their Java software runs on all platforms, it means exactly that; Macintosh included!
