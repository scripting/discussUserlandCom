Certificate support seems to be the same as IE 4.5:

  • You can't add a single certificate, as in Netscape.
  • You can add a certificate authority (CA) if the certificate is in DER format. For example, ours is at http://www.invantage.com/cacert.der.
  • Trying to add a CA cert in PEM format gives "certificate invalid"; Netscape accepts both formats.

I agree that 96 dpi is evil...

96 dpi is necessary to prevent fonts appearing too small. Netscape 6 will do things like this, too. If you set your base font size to 12 point, most everything works well. (I still have no clue why the base font size on Windows was made 12 point ages ago, it's huge!) If you want consistent font appearance across platforms, use pixel units, not point units...
