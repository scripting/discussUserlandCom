LibGD is a Frontier graphics extension (DLL) for creating and manipulating images in GIF format. It is based on Tom Boutell's GD library. The extension was originally developed by Andre Radke who carried the extension through version 1.0a5. (Andre created but did not release versions 1.0a6 - 1.0a8.) I've taken it over with Andre's consent.

New in this version: 5 new drawing functions and enhanced text support.
