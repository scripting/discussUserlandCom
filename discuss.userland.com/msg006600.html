There's a new version of WebEdit that everyone should be using.

Installing a new version of WebEdit requires thought and care. There are compatibility issues, it's best if you can upgrade your entire team at one time.

This version is stable, we've used it for two months, and depend on the new features. So I encourage everyone who's using WebEdit to get the new version now. And if there are any questions, please post them here and we'll improve the docs.
