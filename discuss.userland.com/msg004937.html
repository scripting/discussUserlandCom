Keola,

The inNewsSite field will be added automatically if you use the sample version of console.root released yesterday to manage your news site. Otherwise you will indeed have to go in and add it manually to the message table.

If a message table has an inNewsSite field, everything in that news site's staticPage table will be scheduled to be re-rendered by adding addresses for these items to system.temp.mainresponder.backgroundPagesToBuild. The actual rendering is then done by mainResponder.background.everyMinute which is called from the scheduler.
