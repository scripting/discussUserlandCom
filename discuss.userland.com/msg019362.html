This reminds of something I read about the "Incompatible Timesharing System" that they used at MIT for a while (I think this was from the original Hacker's Dictionary).

They included a "Kill System" command that would do just that.

Occasionally some newbie would discover the command, try it out, and it would do exactly what it said. And so the system would go down. After a while, because it wasn't a challenge, and because the system was a lot more fun when it was up, nobody bother trying to crash the system any more.

Of course, that assumes a certain level of maturity on the part of the participants, but it's still a good stroy, and it probably qualifies as a parable.
