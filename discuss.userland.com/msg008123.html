Gtk+ supports Windows-style tree widgets with '+' and '-' icons, but not Frontier-style triangles. I don't know about Qt.

Qt natively supports a Windows-style tree and a Motif-style tree. The Motif one is kind of a cross between Windows style and the triangles. Here is a screenshot:

http://www.troll.no/qt/treeview.png

Now, the KDE 1.1.1 file manager (kfm) has a treeview with "flipping triangles". Check out the following screenshot for an example:

http://www.kde.org/screenshots/medium/wmaker1.jpg

The "triangles" are actually pixmaps, so it can look like whatever you want...
