I'm probably going to spend the next couple of days working on Prefs.UserLand.Com. As I make changes I'll list them here.

  • We now allow panel names in pathArgs, the text after the $ sign in the URLs. For example, you can get to the Personal Data panel using this URL.

  • Bookmarks. In order to regain my weblog-writing system, I needed to have a panel of bookmarks that displays on the home page of News.UserLand.Com. I'm implementing it so that it will tie into My.UserLand.Com, and be available for any UserLand.Com member.

    How it works. There is a new panel in the Prefs site called Bookmarks. It has a large-ish text area that allows you to enter a limited subset of HTML. That bit of HTML will be displayed under the calendar on the home page as you're browsing. There will be a button that allows you to add a link to the page you're looking at to the Bookmark list.

  • Appearance. Now you can change the link color, vlink color, alink, and text and bgcolor for all sites hosted on Nirvana. This includes the DG, and Buck's, and News.UserLand.Com, and probably a few others. (XML-RPC.COM).
