It seems important to mention that Judge Kaplan didn't equate code with politcal assassination, but that he was making the point that just because something is expression (code or assassination) is not enough to make it expression alone.

His exact words were:

Computer code is expressive. To that extent, it is a matter of First Amendment concern. But computer code is not purely expressive any more than the assassination of a political figure is purely a political statement.
