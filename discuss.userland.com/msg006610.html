Is the code to add "Topic" in the head of a discussion message published somewhere?

Yes, it's available thru the mainResponder root updates process. You can obtain an updated version of mainResponder.discuss.readMessage by bringing the mainResponder.root window to the front and selecting the Update mainResponder.root... command from Frontier's Main menu. Before you do, please make sure that user.rootUpdates.servers.mainResponder.server is set to "madrid.userland.com".
