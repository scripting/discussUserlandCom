Looking at the docs for tcp.sendMail it seems that all I have to do is set the last parameter "text/html" for those messages that are HTML. When you push a message up to the server and want it to go out with any mimetype other than "text/plain", add a string to the msg called mimeType and set it to the exact literal string you want me to send to tcp.sendMail. The code is in the server now, so you can try it out.
