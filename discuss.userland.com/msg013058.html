XML namespaces can't be used with current validating parsers that only support DTDs (at least not the way most people want to use namespaces). XML Schemas are supposed to support using multiple namespaces. Note that the majority of end users and RSS providers don't need to actually validate RSS files, they don't even need parsers with namespace support. At this early point in namespace and schema support, only a handful of people need to maintain a validator and offer it to others to use (like HTML validators).
