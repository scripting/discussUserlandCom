There's a module that can attempt to correct misspelled URLs:

http://www.apache.org/docs/mod/mod_speling.html

What WebSTAR would call a RAW file is called an .asis file in Apache:

http://www.apache.org/docs/mod/mod_asis.html

Using this module, you probably wouldn't need to use .htaccess files.
