I don't recall seeing too many books that allow you to use anything but the ideas presented in the book, surely not the actual words or content. With TrulyFree source, you can not only peruse the code for ideas, but you can incorporate that code into your work and not feel the need to make your own source open. With GPL'd source, you can still peruse the code for ideas (just like a book), you just don't have the right to incorporate the code into your own work, unless you do make your source open as well.

It sounds like TrulyFree and OpenSource works are providing much more freedom than is common in library books.
