All the changes in this version are improvements and fixes for Windows, so Mac user don't have to upgrade (since I have Virtual PC, I can now test and fix my suites under Windows :).

Here are the last changes:

o Displaying the remaining time before the next "auto-saving" and the times since the last savings are fixed.

o I improved the installer so, under Windows, the menu will no longer have items that are useful only on Mac.

o I updated the readme.

You can get autoSaver from: http://www.mose.com/flip/frontier/autosaver.html

AutoSaver makes sure you don't forget to save regularly the Frontier.root and optionally the open GDBs, by checking what you're doing and prompting you to save when you didn't for some (definable) time.
