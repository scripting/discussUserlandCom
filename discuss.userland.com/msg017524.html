OK, so here's a first-cut at how the validator for this might work.

validator1.countTheAngleBrackets (string) returns string

This handler takes a single parameter, a string, that contains any number of left and right . Your handler must return a string, a sentence in exactly this form: There are 12 s. To validate, the numbers must be correct and the angle brackets correctly encoded in the response, according to the XML 1.0 specification.
