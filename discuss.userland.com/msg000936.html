Search Engine XML-RPC Interface

You're invited to add your pages to the Search Engine via XML-RPC.

The interface is straightforward: it takes five parameters, the title, URL, and text of a page, the name of the site the page is in, and the URL of the home page of the site.

The interface is documented on the Developers site:

http://developers.userland.com/stories/searchEngineInterface

Since it's XML-RPC, people who work in Perl and Java and other net-aware environments can add pages as well as Frontier users.

This is the place to talk about it! Let me know how it goes, or if you have any questions or suggestions.
