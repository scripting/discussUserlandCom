LinuxPPC (RedHat Implementation for PowerMac) runs on the iMac.

Pro: http://www.linuxppc.org/iMac/

Con: http://www.linuxppc.org/faq/#compat

Q: Can I run software (applications, binaries) for Intel-based Linux on LinuxPPC?

No. Programs compiled for Intel hardware (or any other processor) will not run on PowerPC-based Linux. If you can get the source code, you may be able to compile the program on LinuxPPC. If the source code is not available for a program, then you'll need to contact the group or company that developed the software and ask if they're planning on making a PPC version.
