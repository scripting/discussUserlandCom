If you are using the shortening URLs technique (<http://frontier.userland.com/tree$3.2.6.3>), you can pretty easily set up your own robots.txt file. Of course you have to have access to the top level of your web servers site. robots.txt only works at the top level of a website. For example <http://my.site.com/robots.txt>

Here's what I did. In my scratchpad.mySite table, I created an wp text object named "robots.txt".

In there I typed in...

# go away
User-agent: *
Disallow: /

That's it!!!

There are tons of sources on how the robots.txt file works. Here is one <http://info.webcrawler.com/mak/projects/robots/robots.html>.
