Trademarks are not an issue as long as you're using the trademark to refer to the genuine article, which you would be if you were using a trademarked that contains a trademarked domain.

UCITA is scary on so many different levels that it's scary.

p.s. I recommend "Software Development: A Legal Guide" by Stephen Fishman, if you want to get up to speed on intellectual property issues (patent, trademark, copyright) as the affect technical people.
