I set it up so that when the old search engine gets a discussion group message to index, it also sends an XML-RPC message to the new search engine machine. This way the Discussion Group is indexed both places, on both servers, by way of "chained" XML-RPC calls.

It also verifies that XML-RPC works when Frontier is running behind IIS.
