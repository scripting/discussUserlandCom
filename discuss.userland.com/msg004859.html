The first thing to do is look in your daily guest database log -- there's a table called Search Engine Client. Each time something is checked in via WebEdit, it either gets indexed or not. If not, the log tells you why.

So -- if you look in the log, you'll get some major clues. Report back here, if you would, if you have trouble figuring out what the clues mean or how to make it work.
