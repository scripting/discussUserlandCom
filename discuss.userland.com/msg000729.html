A question came up on the XML-L list, where are the cool XML demos? I said..

UserLand's XML apps are here:

http://www.scripting.com/frontier5/xml/experiments/scriptingNews.html

http://www.scripting.com/fatPages/suites/siteChanges.html

http://discuss.userland.com/msgReader$598 (Discussion group in XML)

http://discuss.userland.com/msgReader$281 (Folder traversal in XML)

All of these applications are dynamic, meaning that they change. The Scripting News XML file changes whenever the HTML version changes, for siteChanges, it's rebuilt every night at midnite to reflect changed files on the server, and the Discussion Group and Folder Traversal apps are real-time, the XML is computed on the fly on the server, when messages or files are added or modified.
