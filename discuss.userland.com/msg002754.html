Most search engines already index the META NAME="KEYWORDS" information in web pages for just this purpose -- to add words not necessarily found in the document to the list of words the page is associated with.

Even search engines like Index Server that people use on their NT servers do this.

In Frontier, you can easily create KEYWORDS meta tags for pages with the #metaKeywords directive.

Ie:

#metaKeywords "clinton,cnn,byrd"
