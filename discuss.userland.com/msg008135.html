A low tech solution is to install a netatalk server for the Mac and a samba server for the PC on the Linux box. Then you just ask Frontier to save the file on the shared volume, and, inside a telnet session, you ask gcc to compile the file. Ok, its not fun as having back the result in Frontier, but it work fine.
