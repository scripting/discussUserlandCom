What does it do with tags like BLOCKQUOTE?

it removes them, since blockquote tags are not allowed in the RSS and scriptingNews formats...

(I suspect this is not really what you want, but if the mailing list comes up with a format that supports more HTML constructs, I promise to update the tool :-)

And what platform does it run on?

all possible platforms. really. you just need a working Python 1.5.X installation (available from http://www.python.org), and a text editor to tweak the script to suit your needs.
