In IE 4 one can call public methods of a Java applet, so there is no need to create a COM object. I believe this works on both the Mac and Windows. You might need to put the 'MAYSCRIPT' parameter inside the Applet tag.

I tried this and it works on all version 4 browsers on all platforms except MSIE 4.01 on Mac. Maybe it works with IE 4.5 or some newer Java VM? If somebody wants to try, I put a test page with a little scriptable applet at http://helma.at/hannes/test.html.
