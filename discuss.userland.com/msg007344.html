OK - here's some requests for a new RSS/OCS/Scripting New format:

    - What platform is this file for..........? (this parameter would say whether the file was intended for PDA, Web, CD ROM or Broadband - perhaps there could be more - like pre-4.0 browsers versus post-4.0 - Netscape versus IE)

    - Media embedded - this is a major, major improvement. Right now these file formats let in a single graphic - but needless to say should also include a video snippet, an audio snippet and perhaps an animation (either vector or bitmap based)

    - And of course we want all those other extensions - like actually something else besides just the links....... :-)
