I've wanted this since I started using Pike to do Scripting News.

There are scrollup icons on each top-level item. Click on it and you go to the permanent link for that item.

It's done through pikeRenderer.theRenderer, which I modified on the Scripting News editorial server, but the change should probably be distributed soon.

There's a new rule, called internalLink. Its value is the URL of the permanent archive for the page. It also generates the tag.
