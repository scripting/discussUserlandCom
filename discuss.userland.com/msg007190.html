I want to get some advice.

Hypothetically speaking..

Suppose I had $15K burning a hole in my pocket.

I want to buy new server hardware with this money.

Should I buy fifteen $1K machines?

Or thirty $500 systems?

I obviously don't need monitors, keyboards, mice or sound cards for all these systems.

They must be running Windows NT and Frontier, obviously. Fast Ethernet. Intel not Alpha.

Do any companies specialize in selling these kinds of systems?

Ideally I'd like them to be rack-mounted in a reasonable-size enclosure. Using Timbuktu, if possible, to access the desktops of each system. Maybe even an ultra-fast LAN inside each box?

Basically I'm curious what kind of hardware $15K can buy.
