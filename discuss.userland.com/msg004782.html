The Boolean Search Engine is available at http://einstein.stcloudstate.edu/booleanExtension/.

New in this version: a callback to display the search results. Now you can configure the output as well as the input and the form processing. You can even make the little boolean engine *look* like the heavy duty Nirvana engine. See

http://einstein.stcloudstate.edu/booleanExamples/mrsboolSearch.wsf.

The engine has been a fun project and a good learning exercise. This should be the last update for a while, except for bug fixes. Enjoy!
