For a non-alarmist example of what might happen to the browser check out the much vaunted AOLServer, actually a rather nice and fairly well used web server using TCL for scripting. Philip Greenspun (of Photo-Net, Travels with Samantha, and free web tools fame) uses it extensively and it looks like an awesome little platform, beating Apache on many fronts.
