I've released a beta of XML-RPC client classes for PHP. You can get them from http://www.heddley.com/edd/php/xmlrpc.html

This is in early beta, so expect some trouble somewhere along the line. Included in the archive is some documentation and an example of how to query Dave's server on betty.userland.com.

For the moment, PHP's incapable of being an XML-RPC server as it doesn't allow scripts access to the raw POST data. It may be a feature in future versions of PHP.
