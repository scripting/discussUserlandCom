Dreamweaver is an ftp client. The connection could work this way, but http (XML-RPC) is a two way communication mechanism, where ftp is not.

Dreamweaver would be really powerful if say, when you edited a link, XML-RPC talked to frontier, downloaded the glossary list, and presented you with a list of glossary items. You could even add links to the glossary this way. All through Dreamweaver.

This is very do-able if Dreamweaver could XML-RPC. The possibilities are endless.
