This feature is for people who use the Updates page a lot.

Now you can set a preference that limits the display to sites you're interested in following.

If you go to the Prefs page and scroll to the bottom, you'll see a new section called Updates, which has a single item, a text area where you can list excluded sites. List the site names, separated by commas. The site name is the name as it appears in the Updates list. So if you wanted to exclude Scripting News and Dave.EditThisPage.com, you'd enter:

Scripting News,Dave.EditThisPage.Com
