Hi,

Here are the latest changes in AutoSaver:

  • It didn't save the hidden GDBs under Windows. This is fixed.
  • Improved the reporting so if an error occurs when trying to save a file, the error is reported (in scratchpad.AutoSaver_lastErrorsLog) and the process continues.

AutoSaver is a suite (for Frontier 4 and later, Mac and Win) that makes sure you don't forget to save regularly the Frontier.root and optionally the open GDBs, by checking what you're doing and prompting you to save when you didn't for some (definable) time.

http://www.mose.com/flip/frontier/autosaver.html
