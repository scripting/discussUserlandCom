betty.rpc.client("12.34.56.78", 81, "helloWorld", @paramTable)

The server has no clue what to do with it -- it has no idea that it's intended for the XML-RPC server page (which is located somewhere in the IIS wwwroot structure, say at dir1/dir2/Server.asp) to handle.

Try something like this:

betty.rpc.client("12.34.56.78", 81, "helloWorld", @paramTable, rpcPath:"dir1/dir2/Server.asp")
