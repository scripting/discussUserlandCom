GDB subscriptions are complementary to, but not part of, WebEdit. Also: subscriptions expect database names to be the same on client and server.

WebEdit access privileges are on the to-do list, they're not yet implemented.

Question: what kind of access privileges would you like to see? Read and write privileges, for sure, but are you also interested in limiting access by type of object, for instance, or allowing only web pages, or disallowing objects that start with #?
