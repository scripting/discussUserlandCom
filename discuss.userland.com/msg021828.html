And I see that they have a "spy builder" site ... wonder how hard it would be to get it to talk xml-rpc or some other integration...

http://developer.spyonit.com/index.jsp

Try out the main page first to see how it works.. It woudl be interesting if spyonit could Instant message you via the RU instant messaging mechanism - like it can for AOL IM, Microsoft IM, etc. etc.

http://www.spyonit.com/Home

http://www.spyonit.com/Add?_spyid=morbus_CamWorld.com
