I think every one of these apps supports the openURL verb, to which you should be able to pass a local file path.

Of course if you pass the openURL to the Finder, it will look up the users preferred MP3 player (or prompt the user to find one) from the Internet control panel and send the file to that. You wouldn't need 'drivers' or even configuration in the client.

Of course, this assumes you are just passing the file to the MP3 player and nothing more...
