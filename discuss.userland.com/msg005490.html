When you are dealing with dynamicly generated pages, what determines whether a page has changed

Whether the mod date of the object representing the page is greater than the mod date of the cached rendering.

Now that we have verbs that allow you to set the mod date of objects, a script can force re-rendering.

Script objects are never cached.
