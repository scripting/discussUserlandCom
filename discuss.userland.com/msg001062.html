Topic:Search Engine Indexes Edited Pages

The search engine now re-indexes pages that have been edited.

Before, it used to only index new pages: now you can edit a page and update the search engine.

Do be aware that the search engine does cache results, so if you don't see your edit right away it will appear the next time the cache is cleared. (The cache gets cleared a few times a day.)
