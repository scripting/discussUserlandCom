I like and use Ektron's eWebEdit visual editing control (http://www.ektron.com/ewebedit/) to provide Edit This Page functionality for users who want to work with formatted text without dealing with HTML. See their page for a live demo. It still has a few rough edges, and is IE-only until a release targeted for mid June. But still pretty useful.

It comes with back end scripts for asp and Cold Fusion, but these are probably reasonably portable to Frontier.
