Tcp.httpClient defines an adrHdrTable parameter. Could the cookie be stored in memberInfo, then sent in a constructed adrHdrTable? Where can I find the definition of that table?

The name of a table entry corresponds to the name of the header, the value (a string) of the table entry corresponds to the value of the header.

tcp.httpClient already has a facility for dealing with cookies. Check out the cookiesOn parameter and the user.webbrowser.cookies table.
