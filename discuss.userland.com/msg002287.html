> We'll probably want to have a solid spec for how to handle that > because Perl doesn't have booleans at all.

Nor does Python. Fredrik solved this problem by defining a Boolean class in xmlrpclib and two instances, True and False. Booleans in the XML get mapped to those two objects. If Python acquires a boolean type in the future those two symbols can just get set to the true and false booleans.
