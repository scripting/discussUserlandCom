How to use them to generate content filled structures - which the discussion here was about - is of course not covered directly but there is an example "The following example illustrates how the architectural forms defined in this standard could be combined with the hylink and varlink element type architectural forms defined in ISO/IEC 10744:1997 to create a (simplified) document type definition for a navigatable thesuarus/dictionary"
