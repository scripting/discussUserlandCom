Anyone have been using the perl module HTML::Mason and care to comments how it compare to Frontier?

Mason is a fully componentized system. Web pages are built from one or more Mason components. Each component may contain several subcomponents. A component may consist of static HTML, text or graphics generated dynamically by a Perl subroutine, or any mixture of the two. Components can take arguments, and have the ability to process CGI parameters, as well. - Lincoln D. Stein
http://www.webtechniques.com/archives/1999/10/webm/

http://www.masonhq.com/features.html
http://www.perlmonth.com/features/mason/mason.html?issue=9

It look like an interesting CMS but not elegant as Frontier (but I just perused the doc). It must be very powerful since it was developed by CMP Media and used on their own sites like TechWeb and InformationWeek.
http://www.masonhq.com/sites.html

It has also remote management system:
