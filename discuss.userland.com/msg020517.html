The deterministic symbol space is the tuple (NamespaceURI, LocalName) which forms a universally unique element or attribute name

footnote: it's not always a tuple, though. for some XML formats (including WebDAV), you should instead concatenate the NamespaceURI and the LocalName.

that is, "" in the "http://userland/dave" namespace is the same element as "" in the "http://userland/" namespace. but not in all XML-based formats...
