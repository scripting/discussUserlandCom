First you consider a frontier table as an object. then you implement a special metaobject table where you put in frontier functions ( methods ) and as many frontier intrinsic objects (tables, records, lists, strings, binaries, etc... = properties )as you like. From this moment on frontier is an first class agent_Language. And tomorrow....... we implement a XML based GUI on top of it all.
