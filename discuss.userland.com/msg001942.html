Syntax highlighting would be a welcome feature for many people, myself included. It's definitely on the wish list. Not a trivial feature to throw in, however.

Your tooltips suggestion is excellent. On the Mac, we've offered something similar via an option-control-2click, which does a DocServer lookup and expands out the calling syntax in place. Using DocServer, especially over the web, is probably too slow to present the tooltip UI cleanly on most systems. But there's really no reason we shouldn't be able to use the verb's glue script for this information, so the feature should be doable.
