I just put up a new version of the XML-RPC library for Java.

http://helma.at/hannes/xmlrpc/

Among other things, it now contains the XML-RPC applet package, support for XML-RPC proxies, and there are classes for binding XML-RPC into FESI, an ECMAScript interpreter written in Java.

The proxy server is very handy for applets that can only connect to the server they came from. The concept is described and demoed here: http://helma.at/hannes/xmlrpc_test.html
