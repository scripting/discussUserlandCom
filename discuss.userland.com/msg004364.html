It is possible to add Frontier to the "Edit With" button using Mr Domina's method. OMM, this action blanked the "Edit" context command in Explorer, but it was easy to restore it (to Homesite 4.0 in my case).

However, the "%1" sends the path to the file you're reading - in your cache - and AFAIK Frontier expects a folder path, not a file path, in the command line. Are there any command line parameters for Frontier?

Flash: It's a lot easier (and more fun) to use a Javascript bookmarklet, to send the address of the current page to Frontier's webserver.

"javascript:location.href=('http://myServer/myResponder/'+document.location)" does the trick for me.
