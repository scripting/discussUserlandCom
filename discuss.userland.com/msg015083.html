The best index of RSS files is at xmlTree.

You can get a web based categorised listing of over 1200 RSS channels by using this link.

If you want something machine-readable, have a look at the OCS file that xmlTree produces. OCS (Open Content Syndication format) is something I put together a while back to describe channel directories. You can read more about it here.
