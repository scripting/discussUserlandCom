Also, is extraInfo an address or an object? Defaulting to nil implies an address (to me anyway). If I'm passing a table in extraInfo, wouldn't passing the address be more efficient? Maybe it should be adrExtraInfo or adrCallbackData?

Sorry if this seems like a nitpick.
