I was looking around for sample code that does an HTTP POST, but found nothing, so here's a bit of code that sends a request to the What's Related page, and displays the resulting HTML in a wptext window.

local (s = tcp.httpClient ("POST", "nirvana.userland.com", 80, "/whatsrelated/default", datatype:"application/x-www-form-urlencoded", data:"url=http%3A%2F%2Fwww.netscape.com%2F"));
wp.newtextobject (s, @scratchpad.s); edit (@scratchpad.s)

Preserved for posterity.
