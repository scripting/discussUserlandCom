Are there any Frontier scripts available that would help create an RSS file for a site?

Frontier 6 news sites have the ability to automatically generate an RSS file from the contents of your daily news page.

Do items in the odb have date/time stamps or changed item flags that would allow a script to accumulate references that could be packaged up as entries within the RSS file?

Yes, window-based objects like outlines and wp-texts have time-created and time-modified stamps.
