Scalable Content Tool Systems

Author:Marc Canter

Topic:InfoWorld: The New Web Order
Msg #:2742 (In response to 2726)
Prev/Next:2741 / 2743

True Scalable Content Tool systems can be put into place using this approach to Content Management systems. These systems can output not only web sites, but also CD ROMs and Broadband networked versions - of the same content.

This is a critical step in the evolution of profitable business models for content developers. Now the huge investment necessary to develop interactive content - can be spread out and amortized across a wide range of distribtion platforms.

What's most exciting to me is the development of a new paradigm of what 'A Tool' is. No longer single user applications, Tools take on a new meaning and functionality when they are used in a distributed computing environment, utilizing the web and Frontier 6.
