For those who might be interested, Caucho <http://www.caucho.com> has arguably the best Java Servlet, JSP, XML, and XSL-based publishing tool available, called Resin. Check 'em out.

Of particular note, from their "future" page:

"Quercus 1.0 will be a collection of useful Java libraries, XSL stylesheets and scripts. It may turn into a full-fledged content management system like Zope.

Possible features:

  • Weblog support. Make it easy for a weblog writer to create content only using the browser.
  • File upload and multipart-mime.
  • Form library
  • XML-RPC/SOAP
  • Simple graphics library. Querying image file sizes. Graphs and creation of simple gifs. SVG support is possible with Java 2. GIF/PNG?
  • Discussion support.
  • Chat/ICQ support?
  • Simple search engine."

Sounds like an evangelism opportunity!
