How about breaking this survey into 3 parts: use of Open Source software, Open Source license choice, and patent issues.

  1. I create and distribute Open Source software.
  2. I use Open Source software in my solutions. (Example, using MySQL and PHP4 for web-apps)

(If you answer 1 or 3 please also answer questions 2 & 3)

  1. GPL
  2. OSI compatible license (MPL, etc.)
  3. BSD style license
  4. Artistic license
  5. Copyright statement only
  6. Without any license or copyright

As an Open Source developer do you believe in patents?

  1. Yes
  2. No
