Shared menus on the mac are a great thing. When I check out one of my URLs, OMM I can chose from a set of scripts in my Browser's shared Menu, including

  • "Open this page in Frontier",
  • "Reveal local version of this page in Finder",
  • "Reveal server Version in Finder",

and lots of other goodies, that are very easy to write. Now, I don't have a frontier client-server environment, but it should be pretty easy to hook these scripts up with webEdit.

Once again, Mac only.
