This is a feature of the DOM extension (http://www.tallent.com/frontier/dom/). It has verbs that do that (and keeps up with the numbering).

If the variable, origNode, is the address of the Node you wish to duplicate:

local (newNode, nextNode); newNode = DOM.Node.cloneNode (origNode); nextNode = DOM.Node.getNextSibling (origNode); newNode = DOM.Node.insertBefore (nextNode, newNode, nextNode); return newNode;
