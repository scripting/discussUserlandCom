At long last, Tonto is nearing completion!

http://macrobyteresources.com/scripting/frontier/tonto/

In case you're not familiar with Tonto:

Tonto is a small helper application for Frontier. It puts up a small floating window, which displays buttons in any application for activating Frontier scripts that you specify.

The floating window is available in all applications, and the buttons shown in the window can be global (all apps), context-sensitive (specific to the frontmost app), or a combination of both.

Changes in this version include:

  • register lists of scripts all at once (very fast)
  • disable/re-enable window updates (for speed)
  • more icons from James Spahr
  • three different caption sizes (small, meidum, large)

See http://macrobyteresources.com/scripting/frontier/tonto/changes.html for more information on this release.
