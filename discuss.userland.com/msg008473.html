The problem: Jeff Cheney, webmaster at Mac Publishing wants to include MacWEEK headlines in the MacWorld home page. They're already producing a RSS version of MacWEEK news. The MacWorld site is generated with Frontier (as is the MacWEEK site).

What's needed: A macro that reads a channel and renders it as HTML text. The macro is called from the template for the MacWorld home page. It caches the resulting text so that the channel is not read more than once an hour.

The script

Here's a source listing for the macro.

To include it in a page call it as follows:

{rssChannel ("http://macweek.zdnet.com/macweek.xml")}

The rendering

It's very simple. You can tweak it to make it how you like it. (That's why it's sample code.)

You can use this to render any valid RSS file that's accessible over the Internet.
