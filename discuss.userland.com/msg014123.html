From the Ultimate Bulletin Board tour: "After a user selects a particular forum, this screen appears, which lists the most recent topics (those topics that have been posted to within a certain time span). Folder colors next to each topic indicate whether the their have been any additions to a particular topic since the last time the user visited the BB. Users may also conduct a search within the forum or in all forums."

Check out this implementation on a BB I frequent. Notice the little drop-down menu in the upper right of the screen that allows the user to specify a time span.
