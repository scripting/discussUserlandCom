I've received so much help from the Frontier community that I wanted to give something back. So, I'm making available a macro I wrote which might be of use to some people.

The macro generates a full-page HTML calendar, spanning an arbitrary number of weeks. The macro call can include the addresses of one or more ODB tables which hold data on events to be listed in the calendar's cells. (These data tables follow UserLand's mainResponder conventions for calendar-based data structures.)

I use the macro to generate a full-semester academic calendar for each course I maintain a website for. I have one ODB table for university-wide calendar information, and another for each course, holding course-specific dates. In the resulting calendars, I set the general info entries to appear in blue text and the course-specific ones to appear in red text. Cell colors, text colors, cell sizes, and more can be specified in the macro call.

For more information or to download, see:

    http://www-perg.phast.umass.edu/IDB/frontier/calendarScript/

Have fun!
