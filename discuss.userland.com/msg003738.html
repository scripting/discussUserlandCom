As Marc showed, I don't think there's anything required with the HTTP header to use other languages in HTML.

From HTML4.0 Spec: "One important step has been the adoption of the ISO/IEC:10646 standard (see [ISO10646]) as the document character set for HTML."

I guess the only "issues" are generating the characters and having a brower that has a character set to display the page.
