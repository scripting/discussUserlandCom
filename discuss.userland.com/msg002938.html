I you want mainResponder to serve GIFs with a .gif extension, then you need to add that extension to the actual image name in your #images table.

The same goes for serving HTML pages with a .html extension -- the actual object names need the extension. (See http://www.userland.com/ for a dynamic site that works that way.)
