MacOSX shouldn't be that difficult a port as it's already UNIX based.

People have been saying this a lot lately, but it's not necessarily true.

Mac OS X and Unix are only the same at the very lowest layers. For anything with a real human interface, they diverge greatly: Most Unix variants use the MIT X Window System for their interface, while Mac OS X has its own (extremely capable -- moreso than X-Windows -- but different) display and window system architecture. Since that's the part that a user interface deals with, and since a web browser is by definition a user interface...
