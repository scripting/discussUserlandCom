All my X boxes are configured to run the window manager of my choice, and, separately from my window manager configurations, all of my applications have color and similar preference information set via X resources. As long as I log in from something that can see my home directory on the file system those selections follow me around. For the home/work/laptop (separate home directory on each of these three networks) issue there are a small number of preference files that I mirror between machines to get a consistent look.

This is, of course, beyond the simple ability to run an app on another machine and have it display in the desktop of whatever screen I'm sitting in front of at the moment.
