There are at least two possible approaches to this (assuming you have access to your own DNS):

1) go about it like cobalt does, using perl scripts to modify the settings files, then restart the nameserver.

2) Figure out how to use the dns update functions of BIND 8 to do it. There are some in band dns options that can update machine ip addresses or add new machines within a domain. Windows 2000 already tries to send the update packets for machines on DHCP.
