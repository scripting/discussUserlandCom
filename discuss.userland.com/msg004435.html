That's great! You can do that in Netscape 5 too via XUL, in fact change the whole browser interface if you want, and it's cross-platform. Some of the first results are available here:

http://www.mozillazine.org/talkback.html?article=426

There aren't a lot of docs yet, of course the source is available for those really interested people ;-) but one of the main advantages is that you don't need to be a C++ hacker just to change that annoying toolbar in the bottom right of the window, you can just rearrange text in a file, or better yet use an XML or graphical editor. Create a new window in the browser and it'll use the new format. The code attached to the XUL can be JavaScript or C++, or anything else that gets hooked up, that's built into the browser or added on.

http://www.mozilla.org/xpfe/
